Alien RPG Core Rules Module Patch V2.0.4


This is a modified import script for Foundry VTT Version 8.6.
This script will not work under Foundry VTT Version 7.10.

This script will correct the following issues:
  • V9 compatibility fixes
  • Corrected bad weapons links in the WEAPONS AND ARMOR journal entry.

If maps are active or contain tokens,lights or notes they will not be replaced as I don't want to overwrite any work you have done.
You will need to import these manually when convenient.
